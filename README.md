# gizmatron

## Prequisites

Currently Gizmatron runs on the raspberry pi 3 B+. There are plans for this to run on more recent platforms as well

* Raspberry pi os lite Bookworm 64 bit
* Linux Kernel 6.6.20
* Docker
* OpenCV 4.9
* Golang 1.22 arm64
* GoCv latest
* github.com/warthog618/go-gpiocdev
